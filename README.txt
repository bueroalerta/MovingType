-------------------------------------------------------------------------------
MovingType

An animated alphabet.
Built and created for a 2 days workshop with 1st year students at SUPSI, Lugano
Adapted from an older AS3 sketch
by
Andreas Gysin
2009-2012

-------------------------------------------------------------------------------
Purpose

This quite simple program permits to load image sequences from keystrokes.

-------------------------------------------------------------------------------
Examples

A few examples are included with the application: 
use keys F1 to F12 to load different font sets.
-------------------------------------------------------------------------------
Future

A WebGL port is planned (maybe)
